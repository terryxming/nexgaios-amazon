## 词级作战表（三报告联动）方法教训

初版（单月 2026-06）被两个子代理 + 复算证伪，核心坑：

- **别用单月**（尤其淡季）：2026-06 是全年谷底，大盘总成交比 2025-12 小 **7.4×**，机会单被系统性缩 7×。必须多月/稳定篮子或旺季口径。
- **价带闸别用购买中位价**：它是被便宜段压弯的混合中位（karaoke machine 大盘 $60），会把"我们已在其中卖 $300"的溢价尾判成"价带低→放弃"。改用**大盘点击中位价 + 我方实际成交中位价**。
- **竞品只能按 ASIN 归因**：热门搜索词"前3品牌"列漂移（同一 ASIN 24 月被贴 38 个品牌名），品牌名仅参考。
- **三报告都没有竞品价格** → "$240+ 溢价子段占大词多少"算不出，只能估。
- **被证伪的错结论**：不是"没词可攻"，而是"用最淡月 + 便宜价尺子把 30% 的成交池误判死刑"。真正增长在"已站着却没使劲"的大词（免费 CRO 抬份额）+ X2 增长引擎 + SB/DSP 再营销 + Bundle 桥价差。
- → 已固化：**作战表·定版**（滚动12月2025-07~2026-06，修 4 项：品牌词过滤/CRO剔除已领先词/趋势薄样本护栏/价格右尺换单ASIN成交价）。产出 `品牌分析/仪表盘/词级作战表_定版.csv` + `_老板决策版.csv`。核心结论：零广告CRO主攻 4 词（karaoke machine 最大池）、"加投主攻"=0、守·领先词 14、儿童=产品缺口 20 词、头号对手 CYY(B0D1VJMSQG)。**待决**：是否固化进 V3 的 Q4/Q5。


## Q1 市场诊断（CVR"背离"）修正（经三报告审查纠正）

初版诊断被子代理证伪、复算确认，修正结论（已写进 07 切片解读+审计层）：

- **"搜索涨、成交跌、CVR 2.52→2.06" 主要是 2026 年 2-3 月一次性搜索灌水**（2月1.62%/3月1.84%异常低，4-6月回~2.4%），**不是结构性萎缩**。→ 教训：**CVR/背离类结论必须逐月看、别信 H1 平均**。
- **成因三分**：同词转化↓ + 词构成变 + **新词稀释**（约41万搜索、CVR仅0.45%、占跌幅~44%、83%集中在2-3月）。旧"儿童泛需求灌水"假设被证伪（儿童仅占增长8%、CVR高达4.8%）。疑与 Amazon Rufus/AI 搜索泛匹配有关，**待查不咬死**。
- **我们自己健康（搜索目录绩效交叉验证）**：5主力 H1 成交+32%、营收+27%、自家漏斗CVR 3.09→3.65%（升）。→ 教训：**判"我们健不健康"用搜索目录绩效最直接**。
- **唯一真内部警报=CTR 下滑**：曝光+36%、点击仅+12%，CTR 1.20→0.99%。"$300机器摆进$60货架、被看到没被点"，**免费可修**（主图/价格呈现/评分）。
- **karaoke with lyrics 是护城河机器词**（X2是该词赢家榜第2），别当垃圾内容流量弃掉；youtube 才是内容/低价靶子，两词别一锅端。
