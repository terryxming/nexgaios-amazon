**4 个父体 / 变体构成：**

- `B0DHZM8P16`（Break/X 家族）= 4 子体：X2 + X1 + B0DH1P8M7M + B0DP3295MS。Break 家族整体体量最大（约 $1.7M/季，是 S2 家族约 2.4 倍）。
- `B0F4QBR9SV`（S2 家族）= 3 子体：S2 + B0D6Z2228K + B0FDB4BGHG。
- `B0DCZQX11P`（S1）= 独立单体。
- `B0FWBJ9SNP`（S3 家族）= 3 子体，除 S3 外近乎无量。

**产品成本约束**：产品带屏 + 内存条，内存这两年涨价 → 被迫**持续涨价、几乎无向下调价空间**。价格不是可拧的旋钮，打法走价值竞争 + 选择性杠杆（Coupon/Bundle 而非降价），且卡毛利底线。

**利润引擎**：X2（毛利润最高 $74k/量王）+ X1（毛利率最高 36%）。仪表盘旧口径的"Hero"S2 其实是**毛利率最低(16.6%)、毛利润仅第 4、营销最重(约 20% 销售额)的被广告补贴走量品**——病根是自然拉力弱、要靠巨额广告才走量。**不设全局唯一 Hero。**


## 考核单位与 7% 提成口径（已钉死，见 [decisions/0003](decisions/0003-commission-7pct-operator-combo.md)）

- **P&L / 优先级 / 问责单位 = 运营负责人的整体组合**（不是单 ASIN、不是父体）。
- **毛香明**组 = S2 + S3 + 变体：B0DHPN1DMJ、B0FNCQW2K3、B0FDB4BGHG、B0D6Z2228K、B0GVML8C2L（儿童款新品·未开卖）。
- **赵健宏**组 = X 家族 + S1：B0CR1R7FKP、B0CLLT85Y6、B0DCZQX11P、B0DH1P8M7M、B0DP3295MS。
- **7% = 发提成门槛，月度考核**：运营组合整体毛利率 **< 7% 就不发提成**。口径 = 领星 `profit_report` 毛利率（**已含 PPC+DSP+联盟广告**，接近净利）。
- **走量款（S2、X2）刻意压薄甚至略亏**是运营决策（换量/排名/评价入口），靠同运营名下其他 ASIN 把组合拉过 7%。判断"该不该冲量"看**运营组合整体是否守住 7%**，不看单 ASIN 自身毛利。


## 运营组合逐月毛利率（领星 profit_report，准确窗口 2025-08~2026-05）——7% 是紧约束

| 月份 | 赵健宏 (X+S1) | 毛香明 (S2+S3) |
|---|---|---|
| 2025-08 | 5.2% | −2.8% |
| 2025-09 | 2.7% | −1.6% |
| 2025-10 | 8.1% | −1.4% |
| 2025-11 | 8.9% | 2.9% |
| 2025-12（旺季峰）| 12.9% | 7.2% |
| 2026-01（退货潮）| −9.4% | −15.0% |
| 2026-02 | 9.0% | 9.5% |
| 2026-03 | 6.4% | 1.7% |
| 2026-04 | 13.5% | 12.6% |
| 2026-05 | 23.6% | 21.2% |

- 分段：全 10 月 赵 **9.02%** / 毛 **3.95%**；H1-2026 赵 7.03% / 毛 **6.38%（<7%）**；近 3 月(3–5月) 赵 15.3% / 毛 13.7%。
- **月度达标月**：毛香明仅 4/10 月，赵健宏 6/10 月。**1 月结构性必失**——12 月冲的量在 1 月退回（退款率 27–37%），退货尾砸穿当月提成。**退货是一等一利润驱动，SQP 看不到，必须从领星补。**
- **近 3 月回升是主动调整**（收走量强度 + 涨价 + 减促销，用户确认），非淡季自然回血 → 当前姿态毛利可持续。护栏职责 = **防复发**（别在 Q4 又猛冲把整组拖到线下），非事后救火。


## 利润数据源（领星）

- 用 **`nexgaios-lingxing` MCP 的 `lingxing_profit_report`**（dimension=asin / parent_asin，sids=[7481] 香港奥卡-US，monthly，USD）。**不是** `lingxing_mcp` 的 `profit_asin`/`profit_parent_asin`（返回空）。已对齐领星 web 报表分毫不差。
- 利润报表按月查询要求"起止同年月"，**跨月要逐月拉**。准确窗口 **2025-08~2026-05**（DD+7 延迟结算 → 当月偏低）。
- **销售额口径 = `net_amount` = `amount` − `promotion_discount`**（= ASIN360 净销售额）。不要用 `amount`（毛额）。
