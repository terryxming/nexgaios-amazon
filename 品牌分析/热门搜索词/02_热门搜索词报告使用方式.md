# 这份报告能用来干什么

先给一句话的核心判断，再展开。

**热门搜索词是亚马逊唯一「按词点名具体竞品品牌与 ASIN 及其点击/成交份额」的第一方数据。**（注：SQP 也有你自己的品牌份额、也是第一方，只是不点名竞品——本表的独特处是**点名对手**，不是"唯一的竞品数据"。）SQP 告诉你*这个词的市场多大、你吃到多少*，但看不见对手是谁；热门搜索词反过来——它不给你自己的漏斗，却把**每个词的搜索热度排名 + 前 3 名点击品牌 + 前 3 名点击商品(ASIN/标题/点击份额/转化贡献占比)**摆出来。它是一张**竞品格局侦察表**，所有用途都从这个独特性里长出来。

## 一、最核心用途：竞品格局侦察——每个词是谁在吃

这是这张表的杀手锏。对每个重要词，看它的前 3 名品牌和前 3 名商品：

- **Ikarao 在不在前 3？排第几？** 不在 = 这个词你还没占住；在但排第 3 = 有位子但被压。
- **谁是这个词的老大？** ⚠️ **"前 3 品牌"与"点击第 1 商品 ASIN"是两套独立排名、常不是同一家**（实测约 45% 的词，点击第 1 品牌不在点击第 1 商品的标题里）。扒竞品 listing 时**以商品 ASIN 为准**，别默认"品牌老大 = 商品老大"。
- **点击份额 vs 转化贡献占比背离**：⚠️ 先分清——"转化贡献占比"是**成交份额**（该商品成交 ÷ 该词全部成交），**不是转化率**；它和"点击份额"**分母不同（总成交 vs 总点击），不能相减/比大小推转化好坏**。全站结构性常态是"前 3 点击份额之和 > 转化份额之和"（点击集中头部、成交更分散），约 82% 的头部商品会天然显示"转化贡献 < 点击份额"，**照字面把它们判成"成交乏力可抢"就是把噪声当信号**。正确用法：**只看同一个词内商品之间的相对背离**（如点击第 3 名的转化贡献反超第 1 名 = 那个第 3 名 listing 强、值得学或防），别拿单个商品自己的两份额相减。

一句话：**这张表把"谁在赢这个词"从猜测变成点名**，让选品、埋词、竞品拆解有了明确靶子。

## 二、词热度与季节性：用排名升降和词数波动读需求

- **单个词的排名趋势**：跨月看同一个词的搜索频率排名（越小越热）。名次持续上升 = 需求升温，早布局早占位；名次下滑 = 该词降温。
- **新兴词发现**：某个词从无到有、或排名快速蹿升 = 新需求信号（如某种新卖点、新场景词）。
- **月度匹配词数本身**：旺季(11/12 月)卡拉OK相关词数暴涨(240~350)、深淡季(2-6 月)65~86、肩部月(1/10 月)约 100~126，是品类需求季节性的直接量化证据——提前备货、提前布广告的依据。

## 三、给选品与新品线一个「市场在搜什么」的外部视角

- **词里藏着产品方向**：热门词的措辞（adults / kids / with screen / bluetooth / professional…）直接反映顾客在找什么配置。前 3 商品的标题更是把"赢家用什么卖点描述自己"摊开给你看。
- **验证新品线信号**：如果某类词（如儿童相关）持续热门且被特定 ASIN 吃掉，就是"该不该开这条产品线"的市场证据（呼应 Ikarao 已在做的儿童款新品）。

## 四、给广告选词一个「竞争强度」参照

- **前 3 集中度**：如果一个词的前 3 商品点击份额之和很高（如 >60%），说明这个词被头部牢牢把持，新玩家硬抢广告成本高；份额分散则相对好切入。
- **锁定要防守/进攻的具体 ASIN**：知道每个词的第 1 名 ASIN，就能针对性做广告卡位、竞品定向。

## 与其它两张报告的分工

- **热门搜索词（本表）= 竞品视角**：按词看**谁在赢**（排名 + 前 3 品牌/商品）。看不见你自己的漏斗和营收。
- **SQP 搜索查询绩效 = 市场与份额视角**：按词看**市场多大、你的份额、你在哪一环掉队**。看不见竞品是谁。
- **搜索目录绩效 = 自家视角**：按你的 ASIN 看**自己的完整漏斗 + 搜索来源营收**。看不见市场大盘和竞品。

三张表配合：用热门搜索词**锁定要抢的词和对手** → 用 SQP **诊断你在这个词上哪一环掉队** → 用搜索目录绩效**看你的 ASIN 整体成交和营收有没有跟上**。

## 要诚实面对的局限

用之前得知道它**不能**干什么，否则会误判：

- **全表零绝对计数**：没有任何点击数、成交数、搜索量，全是排名 + 份额%——**任何"率/量"都算不出**，别想反推转化率或流量规模。
- **只有排名、且会并列**：搜索频率排名是名次不是量,不能求和、不能算增长率、不能**跨词比名次差**(A 排 3959、B 排 8535 不代表 A 热度是 B 的 2 倍);只能比**同一个词**的跨月名次升降。长尾词排名会并列(ties)、也可达七位数(正常),别假设一名一词或当异常值。
- **只给前 3 名**：第 4 名及以后完全看不到，前 3 份额之和 <100%，长尾竞争格局是盲区。
- **没有你自己的漏斗**：这张表不含曝光/点击/加购/成交的你方数据，诊断漏点要回 SQP。
- **没有营收、没有利润**：看不到成交金额，更没有成本，投放 ROI 要结合领星利润数据。
- **按关键词过滤**：本导出只含 karaoke 相关词，不是全站；换过滤词得到的词集不同，别当全品类。
- **月粒度、月末才出**：反应慢，是战略侦察工具，不适合日常调 bid/调价这类即时决策。

---

**一句话总结意义**：这份报告的价值不在于任何单个数字，而在于它把每个热门词变成一张**竞品点名榜**——你能看清这个词多热、谁在赢、赢家是哪个 ASIN，从而把竞品研究、埋词选品、广告卡位、新品方向这些动作，都**对准到具体的词和具体的对手**上，而不是凭感觉。它和 SQP 是一枚硬币的两面：SQP 看自己，热门搜索词看对手。
