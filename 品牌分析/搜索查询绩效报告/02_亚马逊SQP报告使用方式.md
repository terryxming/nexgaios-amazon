# 这份报告能用来干什么

先给一句话的核心判断,再展开。

**SQP 是亚马逊唯一把「关键词的市场大盘 + 你的份额 + 完整购物漏斗」三者合一的第一方数据。** 它让运营从"我卖了多少"(销售报告能告诉你的)升级到——**这个词的市场有多大、我吃到了多少、我在哪一环掉的队**。这三个问题,别的报告单独都回答不了,只有 SQP 一次给全。它的所有用途,都是从这个独特性里长出来的。

## 一、最核心用途:关键词级的"漏斗漏水点"诊断

这是这张表的杀手锏。因为每个阶段都同时有「大盘总量」和「你的份额」,你能精确定位**在哪个环节掉队**,而不是笼统地说"这个词表现不好"。

对每个重要词,横向扫一遍你的份额链条:

| 症状(你的份额链) | 判断 | 运营动作 |
|---|---|---|
| 曝光份额就低 | 词没占住位,自然排名+广告都不够 | 加广告卡位、埋词优化自然排名 |
| 曝光份额高、**点击份额掉下来** | 展示了但没人点你 → 主图/价格/评分/标题不吸引 | 优化主图、补 Review、调价 |
| 点击份额高、**加购/购买份额掉下来** | 点进来了但没成交 → 详情页/价格/评价/库存劝退 | 优化 A+、变体、价格、清差评 |

一句话:**份额在哪一级"断崖",问题就在那一级**。这把运营优化从"拍脑袋"变成了"照着漏斗找漏点"。

⚠️ **必须先过样本量闸**:只对**大盘购买总数够大(如 ≥30–50 单)**的词做份额漏斗诊断。实测品牌视图里约 85% 的词大盘购买 <10 单、约 1/4 的词份额直接 = 100%(其实是 1÷1)——**小样本词的点击/加购/购买份额是噪声、"断崖"多是假象**,别据此下结论。

## 二、关键词机会挖掘:用"搜索量 × 你的份额"矩阵选战场

把词按两个轴分类,直接得到打法优先级:

- **高搜索量 + 你份额低** → **最大机会词**:市场很大、你几乎没吃到,值得重点投广告去抢。
- **高搜索量 + 你份额高** → **守成词**:你的基本盘,盯着别被竞品蚕食(跨月看份额有没有下滑)。
- **低搜索量 + 你份额高** → **鸡肋/精准词**:量小,维持即可,别过度投入。
- **大盘转化率(第26列)高的词** → **高价值词**:这个词的顾客购买意图强。⚠️ **先按搜索量/购买量过滤**——实测大盘 CVR >50% 的词中位搜索量只有 1(单次搜索→单次购买的偶发,CVR=100% 无意义);不过滤直接按 CVR 排序会捞出纯噪声。

对 Ikarao(卡拉OK机)这种品类,这一步能直接产出一张"该抢哪些词"的清单。

## 三、给广告投放(PPC)一个"市场天花板"

广告报告只告诉你**自己**花了多少、转化多少;SQP 补上了广告看不到的两件事:

- **增长空间还有多大**:你的购买份额离 100% 差多少,就是这个词理论上还能抢的量。
- **行业基准线**:大盘的点击率、转化率就是这个词的"及格线"。你广告的 CTR/CVR 低于大盘,说明是你的 listing 问题,不是词不行;高于大盘,说明可以加预算放量。

这让加词、否词、调 bid 有了外部参照,而不是只跟自己历史比。

## 四、价格定位与配送时效洞察

- **价格带**(第11/12、20/21、29/30列):对比「大盘中位价 vs 你品牌中位价」。你贵一大截却份额还行 = 品牌溢价成立;贵且份额低 = 价格在劝退顾客。
- **配送时效**(当日达/1天/2天那几列):看这个词的大盘点击/购买有多少集中在快速配送档。如果高度集中,说明该品类顾客**对时效敏感**,这是你要不要上 FBA、多仓布货的直接证据。

## 五、趋势与季节性(你有 17 个月,这是最该做的)

单月是快照,你手上是**17 个月的序列**(2025-02~2026-06),价值在跨月对比。⚠️ **但每月 Top1000 榜大幅换血**(相邻月词集只重叠约一半、17 月全在的稳定词仅约 200 个):**跨月对比/求趋势必须先落"稳定篮子"(多月都在的交集词)+ 披露覆盖率,不能对换血的 Top 榜裸求和/裸对比**,否则会把"某词进/出榜"误读成需求暴涨暴跌。

- **季节性**:卡拉OK机在节假日/圣诞的搜索量高峰,提前备货、提前布广告。
- **你份额的月度走势**:上升=策略见效;**下降=竞争加剧的预警信号**,越早发现越好。
- **新兴词爬升**:某个词搜索量连续几个月上涨 = 新需求,早布局早占位。

## 两种视图的分工

- **品牌视图(1000词)= 战略层**:整个品牌在词海里的攻守全景,做选品方向、大盘策略。
- **ASIN视图(100词)= 战术层**:单个 SKU 靠哪些词被搜到、在哪些词上转化好/差,做单品级 listing 和广告优化,能归因到具体商品。

## 要诚实面对的局限

用之前得知道它**不能**干什么,否则会误判:

- **只有 Top 1000 / Top 100 词**:长尾完全看不到,不能当全量。
- **月粒度、月末才出**:反应慢,**不适合日常调价/调 bid** 这类即时决策,是战略复盘工具不是实时仪表盘。
- **看不到具体竞品是谁**:只知道"大盘 - 你 = 别人",推断不出对手是谁、做了什么。
- **份额不分自然/广告**:是品牌整体口径,拆不出这部分份额是花钱买的还是自然来的。
- **没有营收、没有利润,成交列是市场级**:第 25/27 列是"这个词全市场 / 你品牌的成交单数",不是你某 ASIN 卖了多少、更不是赚了多少钱。**判 ROI / 赚不赚钱要去搜索目录绩效(营收)+ 领星(利润)**——本节 §二/§三 的"ROI 更好""值得投"只是相对方向,别当收益结论。

---

**一句话总结意义**:这份报告的价值不在于任何单个数字,而在于它把每个关键词变成一条**可诊断的漏斗**——你能看清市场多大、自己站在哪、从哪一环开始掉队,从而把广告预算、listing 优化、选品、备货这些动作,都**对准到具体的词和具体的环节**上,而不是凭感觉。
