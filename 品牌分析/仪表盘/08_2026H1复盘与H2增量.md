# 2026 H1 复盘与 H2 增量（24 子代理对抗核验）

> 面向老板/高管。顶层白话可扫读，术语与推导沉到「▸ 审计」层。数据锚定 **2025-02 ~ 2026-06**（受 SQP 报告跨度限制）。品牌合并口径 = 香港奥卡-US(7481) + 东莞无界未来-US(12137)。
> 方法：8 个研究单元（V3 六问题 + H1 复盘 + H2 增量）各派 正方/反方/中立整合 三个子代理对抗核验（共 24 个），中立整合逐单元独立复算、纠正双方、标注来源与踩坑。原始会话存档见 [存档/子代理研究_2026H1_20260710/](存档/子代理研究_2026H1_20260710/)。口径边界见 [00 总纲](../00_三报告口径与边界总纲.md)（尤其 §七 三报告 vs 领星不可对账）。


## 一句话总纲

上半年我们**真的把量和份额做上去了，但赢得低质、利润没过提成线**；下半年的增量真实但有限，且**每一条都得绕开三个陷阱——市场其实没在扩、Q4 的退货尾、7% 提成闸**。


## 一、2026 H1 复盘（已结束）

**结论：量和份额做对了，利润质量上半年不及格、但年中已明显回血。**

- **赢（真实增长）**：品牌合并销量 13,758→17,146（**+24.6%**，干净月 4–6 月 +25.9%）；SQP 购买份额四环全抬（曝光 9.00→10.74 / 点击 6.66→7.55 / 加购 4.29→5.11 / 成交 **3.43→4.60，+1.18pp**）；子类目 BSR 全线前进（S2 10→4、X2 3→2）。从 CYY 等对手手里真抢到份额。
- **输（利润没过线）**：H1（Jan–May 可靠区）两运营组合并毛利 **6.87% < 7% 提成线**（毛香明 6.38% / 赵健宏 7.03%）。主因是 **1 月退货潮**——一个月吐掉去年 12 月利润的 **22.3%**（−$145,292），是 Q4 冲量的机械回吐。
- **回血（主动调整、可持续）**：3–5 月靠**涨价 + 收促销**把毛利拉回 **13–15%**（S2 净ASP $245→$297、X2 $332→$393），退货率回落到 8–9%。方向对。
- **唯一 H1 没交出改善的漏点**：**自家点击效率崩**——被看到更多、被点更少（自家 CTR 1.23%→0.89%，−28%）。最省钱却没修。
- **增长质量偏低端**：约 **85% 的增量来自便宜档**（S3 新品 + 非主力变体）；**旗舰 ASP 其实升了**（4 款旗舰净 ASP +7.4%），品牌 ASP −6.2% 是**结构下沉（多卖便宜机型）不是旗舰降价**。

## 二、六个问题诊断（V3 Q1–Q6）

- **Q1 市场与季节**：我们 +25% 增长是真的（两把尺互证、连干净月都成立）；但**"市场需求在扩 +13.5%" 是假象**——全是 2–3 月一次性搜索灌水，干净月搜索持平、成交还跌约 6%，市场 CVR 崩 2.52%→2.06%。**在不增长的池子里抢份额**。季节硬锚：2025 Q4 = 35,298 台 / $10.4M（Dec ≈ 淡季平月 6.7–7×）。
- **Q2 在赢份额吗**：份额升成立（+1.18pp、四环全抬），但**成色打折**——中段点击/加购两环的份额抬升几乎全靠大盘自己塌（分母幻觉）、头条约 ¼ 来自成交池缩水；真实增量押在最便宜的 S3 + 最低毛利的 X2。**别用"缩量市场/对手撤退"叙事（无证据）**。
- **Q3 5-ASIN 角色/集中度**：**废掉"单一 Hero=S2"是对的**（S2 是薄利 17% + 高退货 40% + 最吃广告的走量入口，不是顶梁柱）；真引擎是 X2。但"角色互补 ≠ 风险分散"——**X 父体(X1+X2)占 57.6% 净额、X2 单款 41.5%**，这才是真正的单点/单父体风险，看板挂这个标签，不是旧口径的 S2。
- **Q4 漏斗漏点**：业务口径下最该修的是**曝光→点击（被看到没被点，唯一在恶化、修起来最便宜、天然过 7% 闸）** + 中段点击→加购（卡价格墙）。**别把"漏点"翻译成"加投放"**——下游还在漏，灌冷流量 = 花钱 refill 漏桶（S2"曝光最高、转化最差"是现成反例）。
- **Q5 抢词/守词**：真阵地是**专业/功能词**（professional / best-for-adults，X2 是第 1 点击 ASIN、把 CYY 压在下面）；两个最大通用词（karaoke machine 97.8 万搜索）**我们其实不在前三、CYY 逐月拉开点击份额（28%→33%）**——"站着没使劲"是**品牌名列漂移的误读**，抢它是高成本、不是免费活。儿童词 = 新品立项信号，本季不投现有 $300 机型。
- **Q6 增长打法 + 护栏**：**免费 CRO 杠杆全 SKU 立刻放开**（零毛利成本、天然过闸）；付费必须过**运营组合 7% 月度闸**（毛香明组 H1 仅 6.38%<线）、按 SKU 分档（X1 有空间、S2 最要克制）；Q4 按"冲量月 + 退货尾"联算封顶。

## 三、2026 H2 增量（按 ROI + 护栏排序）

1. **免费修 CTR / 主图**——唯一"无条件净增量"：补曝光→点击缺口 + 74.6% 弃单，零毛利成本、天然过闸。同时抬"曝光→点击"和"加购→成交"两环。
2. **S3 放量**——最好的 NET 抓手：既吃量、又是**毛香明组毛利率的解药**（23.6% > 组 6.38%，帮过 7% 月度闸）、SQP 份额 <1% 头顶全空间、CVR 5.53% 最高。**但认清它救的是"率"不是"额"**（单台毛利额仅 $47）。
3. **守扩功能词**（professional / with-screen / with-lyrics，我方 12–21% + 溢价）——保毛利、不扛量。
4. **Q4 旺季**——全年最大最确定的池（2025 Nov+Dec = 30,201 台 / $8.9M，比整个 H1 还多）。目标设法见下。现金/库存向高毛利 X 家族倾斜。
5. **加购弃单挽回**（免费改 listing + 付费再营销过闸）；**儿童新品 B0GVML8C2L**（未开卖）按产品线节奏——战略卡位、非 H2 立刻变现。

### Q4 目标怎么设（份额侵蚀 vs 上行，待老板拍板）

旺季实锚 = 2025 Q4 = 35,298 台。**关键变量：旺季我们的市场份额会被压（侵蚀）还是保住/上行？** 唯一观测（2025-12 份额 3.06%，低于夏季 4.7–4.8%）指向被压缩，但一个点无法判断是结构性还是 2025 一次性。

- **份额侵蚀（保守·建议为主目标）**：不假设旺季份额上行；Q4 增量主要来自 S3 新增 + 免费 CRO，扣退货后目标锚在 **2025 Q4 量级（~35k 台）持平至微增**。
- **份额上行（乐观·情景）**：若库存 + 广告预算 10 月前到位、份额延续干净月 +25.9% 抢夺轨迹 × 季节倍数，毛口径可外推至 ~44k 台，**扣退货后净 ~34–37k**。

**两种都别做的事**：拿毛 44,000 台当承诺、假设旺季还能像淡季一样抢份额、为冲量深促埋 1 月退货雷（去年 −$145k）。

<details><summary>▸ 审计 · H2 护栏（7% 月度提成闸）</summary>

- **免费杠杆**（CRO/主图/评分/弃单挽回/listing）：全 SKU 无条件放开，零毛利成本、不占闸。
- **付费杠杆**（深 Coupon/Prime/DSP）：按**运营组合 durable/loaded 口径**过 7% 月度闸，按 SKU 分档——赵健宏组（durable 9%）有空间、X1 优先；毛香明组（durable 3.95–6.38%<7%）严格克制、S2 基本不加付费。
- **Q4 硬约束**：预算按"冲量月 + 次年 1 月退货尾"联算封顶、预置退款率；退货只能靠领星监控（SQP 看不见）。
- 7% 考核 = **月度**（Terry 确认，同 ADR-0003）：1 月结构性必失，护栏职责 = 防 Q4 冲量把毛香明组再砸穿。

</details>


## 四、研究抓出的口径纠正（已回写蓝图/文档）

- **删"缩量市场/对手撤退"叙事**（无证据、与定稿口径相悖）→ 用"搜索在涨、成交在缩、我方份额逆势升"。
- 蓝图 Q4 数字漂移：曝光→点击 **3.18pp → 2.88pp**、加购→成交 **25.4% → 24.80%**（以本次第一手复算为准）。
- S3 趋势"连续 4 月上行" → **"3 月起 3 连升"**；"头部流量入口" → **X2 + S2 双入口**（imp_share 3.43/3.41%），非 S2 单一。
- "守词 = 守毛利"是跨报告混口径 → 守毛利回领星实盘。
